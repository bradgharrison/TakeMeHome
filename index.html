TakeMeHome

Chrome extension for quick homepage access via keyboard shortcut.

How it Works

Set a homepage URL and a keyboard shortcut. Pressing the shortcut will switch to an existing tab for that URL or open it in a new tab.

  1. Set your desired homepage URL using the popup (click the toolbar icon).
  2. Assign a keyboard shortcut via chrome://extensions/shortcuts (default is Alt+T).
  3. Use the shortcut to navigate to your homepage.
  4. For even faster access, check the "Global" option in the shortcuts page to make the shortcut work even when Chrome is minimized.

Installation

Manual installation required:

  1. Download or clone this repository
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode"
  4. Click "Load unpacked" and select the TakeMeHome folder

Permissions Required

  • storage: Saves the homepage URL.
  • tabs: Finds, focuses, and opens tabs.
  • commands: Enables the keyboard shortcut.
  • action: Provides the configuration popup.

Privacy

This extension is designed to be privacy-friendly:

  • No tracking or analytics
  • No data sent to external servers
  • All settings stored locally in Chrome's storage
  • No browsing history access beyond what's needed for core functionality
